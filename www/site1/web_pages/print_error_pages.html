HTTP Request Tester

← Back to Home

This page lets you manually send raw HTTP requests to the server through a Bash CGI script. You can test invalid requests, trigger error responses, or try valid commands to check how the server behaves.

Examples

Valid GET

GET / HTTP/1.1
Host: localhost

Unknown method

FOO / HTTP/1.1
Host: localhost

Missing host

GET / HTTP/1.1

Non-existing page

GET /does_not_exist HTTP/1.1
Host: localhost
